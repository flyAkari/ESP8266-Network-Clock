# Network clock and number of your followers monitor

可以在看时间的同时看你有多少粉丝。

## How to use 如何使用

将软件刷写进已经连接好的硬件中，连接 wifi `flyAkari` ，打开网址 `192.168.4.1` ，连上一个可用的wifi即可。

注意：你的手机可能会比较排斥公共的wifi。

按流量计费的用户请注意：这个东西不会消耗很多流量。

## How to conncet wire 如何连线

| esp8266 | 转接板 |
| :-----: | :----: |
|   D1    |  SCL   |
|   D2    |  SDA   |
